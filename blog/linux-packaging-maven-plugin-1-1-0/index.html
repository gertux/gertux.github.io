Linux Packaging Maven Plugin : 1.1.0

2015-10-24

Version 1.1.0 of the Linux Packaging Maven Plugin is released.

Fixes:

  1. #3 : Complete Debian binary package control file handling.
  2. #5 : It should be possible to mark files as config files using an expression.
  3. #6 : Move package resources to the default resources path, this simplifies the use of the standard resources plugin, which is nice if one wants to use resources filtering.

The plugin artifact is published to Maven Central .
