Linux Packaging Maven Plugin : 1.2.0

2018-06-30

Version 1.2.0 of the Linux Packaging Maven Plugin is released.

The main changes are:

  1. support for symbolic links in the package contents
  2. packaging maven dependencies in a folder that can be included on the classpath

The plugin artifact is published to Maven Central and the documentation is also available.
